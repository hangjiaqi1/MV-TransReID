
# MV-TransReID Reconstruction
## Requirements
Python 2.7
TensorFlow tested on version 1.3, demo alone runs with TF 1.12
## Installation
Setup virtualenv
'''
conda create --name hmr python=2.7
conda activate hmr
pip install numpy
pip install -r requirements.txt
'''
## Install TensorFlow
'''
conda install tensorflow-gpu==1.11.0
pip install open3d 
'''
